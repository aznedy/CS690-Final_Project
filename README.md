# CS690 Parking Garage Management System

Console-based parking garage management application for the BSU CS690 course
final project. Built on .NET 10 with [Spectre.Console](https://spectreconsole.net/).

## Implemented so far

| FR    | Description                                                       |
| ----- | ----------------------------------------------------------------- |
| FR-01 | Check parking spot availability on vehicle entry                  |
| FR-02 | Block vehicle entry when the garage is at full capacity           |
| FR-03 | Record vehicle entry timestamp                                    |
| FR-07 | Calculate the parking fee from entry and exit timestamps          |
| FR-08 | Apply an overtime rate when the allowed duration is exceeded      |
| FR-04 | Add / Remove / Modify subscriber data                             |
| FR-13 | Display each parking spot's status (spot-status grid)             |
| FR-09 | Display an itemized fee breakdown                                 |
| FR-06 | Display a report of subscribers including the assigned spot, associated vehicle tag and subscription expiration    |
